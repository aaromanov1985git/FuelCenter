# Alembic Миграции

Эта директория содержит миграции базы данных для проекта GSM Converter.

## Структура

- `env.py` - Конфигурация Alembic, подключается к настройкам приложения
- `script.py.mako` - Шаблон для создания новых миграций
- `versions/` - Директория с файлами миграций

## Создание начальной миграции

Если миграции еще не созданы, выполните:

```bash
cd backend
alembic revision --autogenerate -m "Initial migration"
```

Это создаст файл миграции в `versions/` на основе текущих моделей из `app/models.py`.

## Применение миграций

```bash
# Применить все миграции
alembic upgrade head

# Откатить последнюю
alembic downgrade -1
```

## Важно

- Всегда проверяйте автогенерированные миграции перед применением
- В production применяйте миграции вручную перед деплоем
- Делайте бэкап БД перед применением миграций
