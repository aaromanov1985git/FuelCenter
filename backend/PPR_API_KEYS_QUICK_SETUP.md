# Быстрая настройка API ключа для провайдера

## Шаги настройки

### 1. Откройте шаблон провайдера

В интерфейсе управления провайдерами:
- Выберите нужный провайдер
- Перейдите к шаблонам провайдера
- Создайте новый шаблон или отредактируйте существующий

### 2. Добавьте API ключ в connection_settings

В поле `connection_settings` (JSON) добавьте:

```json
{
  "api_key": "ваш_уникальный_ключ"
}
```

### 3. Сохраните шаблон

После сохранения API ключ будет доступен для использования.

## Примеры

### Провайдер "ППР" (ID: 3)

```json
{
  "api_key": "yzsdzCsjyJpHOHwSwnynQzsGVDEZeXcR",
  "base_url": "https://online.petrolplus.ru",
  "provider_type": "petrolplus"
}
```

### Другой провайдер (ID: 5)

```json
{
  "api_key": "другой_уникальный_ключ_XYZ123",
  "base_url": "https://api.another-provider.com"
}
```

## Проверка

После настройки при запросе к API с ключом `yzsdzCsjyJpHOHwSwnynQzsGVDEZeXcR` будут возвращены **только транзакции провайдера с ID=3**.

При запросе с ключом `другой_уникальный_ключ_XYZ123` будут возвращены **только транзакции провайдера с ID=5**.

## Важно

- ✅ Каждый ключ привязан к одному провайдеру
- ✅ Транзакции фильтруются строго по провайдеру ключа
- ✅ Если ключ не найден → ошибка 401
- ✅ Если провайдер не определен → ошибка 403
