# Важно: Перезапуск сервера после изменений

После внесения изменений в код роутеров или сервисов необходимо **перезапустить сервер FastAPI**.

## Как перезапустить сервер

### Если сервер запущен в терминале:
1. Нажмите `Ctrl+C` для остановки
2. Запустите снова:
   ```bash
   cd backend
   python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
   ```

### Если сервер запущен как служба:
1. Остановите службу
2. Запустите снова

### Проверка что сервер перезапущен:
После перезапуска проверьте, что изменения применены:
```powershell
# Проверка доступности
Invoke-RestMethod -Uri "http://10.35.1.200:8000/api/ppr" -Method Get
```
