# Интеграция уведомлений на фронтенде

## Созданные компоненты

### 1. NotificationsList.jsx
Компонент для просмотра списка уведомлений:
- Отображение списка уведомлений с фильтрацией
- Фильтры: статус прочтения, категория, тип
- Отметка как прочитанные
- Удаление уведомлений
- Пагинация

**Использование:**
```jsx
<NotificationsList />
```

### 2. NotificationSettings.jsx
Компонент для настройки уведомлений:
- Настройка каналов (Email, Telegram, Push, In-app)
- Настройка категорий уведомлений
- Сохранение настроек

**Интегрирован в Settings** в секцию "Уведомления"

### 3. NotificationBadge.jsx
Компонент для отображения счетчика непрочитанных уведомлений:
- Автоматическое обновление каждые 30 секунд
- Отображается только если есть непрочитанные уведомления
- Используется в навигации

**Использование:**
```jsx
<NotificationBadge />
```

### 4. useNotifications.js (хук)
Хук для работы с уведомлениями:
- Получение счетчика непрочитанных уведомлений
- Автоматическое обновление
- Используется в NotificationBadge

## Интеграция в App.jsx

1. **Добавлена вкладка "Уведомления"** в навигацию
2. **Добавлен бейдж** с счетчиком непрочитанных уведомлений
3. **Добавлен роутинг** для компонента NotificationsList

## Интеграция в Settings

1. **Добавлена секция "Уведомления"** в настройки
2. **Интегрирован компонент NotificationSettings**

## API Endpoints

Все компоненты используют следующие endpoints:
- `GET /api/v1/notifications` - получение списка уведомлений
- `POST /api/v1/notifications/mark-read` - отметка как прочитанные
- `DELETE /api/v1/notifications/{id}` - удаление уведомления
- `GET /api/v1/notifications/settings` - получение настроек
- `PUT /api/v1/notifications/settings` - обновление настроек

## Использование

1. **Просмотр уведомлений:**
   - Перейдите в раздел "Уведомления" в навигации
   - Используйте фильтры для поиска нужных уведомлений
   - Отмечайте как прочитанные или удаляйте

2. **Настройка уведомлений:**
   - Перейдите в "Настройки" → "Уведомления"
   - Включите нужные каналы
   - Настройте категории уведомлений
   - Сохраните настройки

3. **Счетчик непрочитанных:**
   - Автоматически отображается в навигации
   - Обновляется каждые 30 секунд
   - Клик по "Уведомления" открывает список
