# Архив устаревшей документации

Этот каталог содержит устаревшую документацию, которая сохранена для справки, но больше не является актуальной.

## Структура архива

### [Отчеты](./reports/)
Исторические отчеты по анализу, диагностике и адаптивности проекта.

### [Планы](./plans/)
Выполненные планы и проектирование, которые были реализованы.

### [Старые troubleshooting](./old-troubleshooting/)
Устаревшие документы по решению проблем, которые были заменены актуальной документацией.

## Примечание

Если вы ищете актуальную документацию, вернитесь в корневую директорию [`docs/`](../).
